
## Athena Migrate
this tool will helo you create athena.yaml file and run migrations for all packages.

### How to use
to run use:
```bash
dart pub global run flutter_plugin_tools [COMMAND]
```


comands included:
```bash
  create    Create a new migration file.
  up        One migration up.
```
